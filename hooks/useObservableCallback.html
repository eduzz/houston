useObservableCallback#

Retorn um callback que dar Subscribe e retornar o resultado de um Observable e quando o componente desmonta (unmount) tambm d unsubscribe, diminuindo assim o risco de Memory Leak. Como useEffect que retorna o valor do Observable

Como usar#

tsx
1
const [callback, value, error, completed, loading] = useObservableCallback(() => userService.get(), []);
2


3
return (
4
  <div>
5
    {loading ? 'Carregando' : ''}
6
    {value}
7
    <button onClick={callback}>iniciar</button>
8
  </div>
9
);

Cuidados#

  • Erros ocorridos sero automaticamentes logados (no tratados) se as o onUnhandledError no setHoustonHooksConfig for setado, mas se utilizar o operador catchError esse erro no ser logado pois foi previamente tratado, nesse caso o log deve ser feito manualmente.

  • O valor inicial ser undefined, lembre-se de tratar isso quando estiver usando:

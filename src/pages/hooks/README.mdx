# Hooks

[![version](https://img.shields.io/npm/v/@eduzz/houston-hooks)](https://www.npmjs.com/package/@eduzz/houston-hooks)
[![size](https://img.shields.io/bundlephobia/min/@eduzz/houston-hooks)](https://www.npmjs.com/package/@eduzz/houston-hooks)

Hooks diversos para auxiliar no desenvolvimento.

### Instalação

```bash
yarn add @eduzz/houston-hooks
```

### Configuração

```ts
import setHoustonHooksConfig from '@eduzz/houston-hooks/config';

setHoustonHooksConfig({
  onUnhandledError: (err, origin) => {
    //adicione seu logger
  },
  pagination: { pageStart: 1, perPage: 25 }
});
```

| config           | default                       | descrição                                                                                                                                                                                                      |
| ---------------- | ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| onUnhandledError | null                          | Para facilitar, adicone o onUnhandledError, assim todo o erro ocorrido (não tratado) nos hooks **usePromise** e **useObservable** será enviados para esse função, **ELA É APENAS PARA LOGAR, NÃO TRATA NADA.** |
| pagination       | { pageStart: 1, perPage: 25 } | Configuração default da paginação dos hooks **useObservablePaginated** e **usePromisePaginated**                                                                                                               |

# Dropdown

Componente suspenso. Pode ser utilizado para criar listas, menus e outros componentes flutuantes.

### Importação

```js
import Dropdown from '@eduzz/houston-ui/Dropdown';
```

### Exemplo

Clique no botão para exibir o conteúdo.

<Playground>
  {() => {
    const Content = () => (
      <div className='content' style={{ width: 400 }}>
        <List dividers>
          {new Array(3).fill('').map((_, index) => (
            <List.Item key={index}>
              <List.Left>
                <Thumbnail src='https://picsum.photos/80' size='md' />
              </List.Left>
              <List.Text title='Lorem Ipsum Dolor' description='Lorem Ipsum Dolor' />
              <List.Right>
                <IconButton>
                  <ArrowRightIcon />
                </IconButton>
              </List.Right>
            </List.Item>
          ))}
        </List>
      </div>
    );
    return (
      <Dropdown content={<Content />}>
        <Button>Exibir conteúdo</Button>
      </Dropdown>
    );
  }}
</Playground>

### Direção

Existem 12 possibilidades para se adequar ao conteúdo.

<Playground>
  {() => {
    const Content = () => (
      <div className='content' style={{ width: 170 }}>
        <List dividers>
          <List.Item>
            <List.Left>
              <EditOutlineIcon />
            </List.Left>
            <List.Text title='Editar' />
          </List.Item>
          <List.Item>
            <List.Left>
              <TrashOutlineIcon />
            </List.Left>
            <List.Text title='Remover' />
          </List.Item>
        </List>
      </div>
    );
    return (
      <Row>
        <Column>
          <Dropdown content={<Content />} placement='top'>
            <Button>Cima</Button>
          </Dropdown>
        </Column>
        <Column>
          <Dropdown content={<Content />} placement='left'>
            <Button>Esquerda</Button>
          </Dropdown>
        </Column>
        <Column>
          <Dropdown content={<Content />} placement='bottom'>
            <Button>Baixo</Button>
          </Dropdown>
        </Column>
        <Column>
          <Dropdown content={<Content />} placement='right'>
            <Button>Direita</Button>
          </Dropdown>
        </Column>
      </Row>
    );
  }}
</Playground>

> Todos os lados possuem mais duas direções. Por exemplo: `top`, `top-start` e `top-end`. Sempre adicionando o sufixo (`-start` e `-end`).

| prop      | tipo                | obrigatório | padrão         | descrição                   |
| --------- | ------------------- | ----------- | -------------- | --------------------------- |
| placement | `DropdownPlacement` | `false`     | `bottom-start` | Posicionamento do conteúdo. |
| content   | `React.ReactNode`   | `true`      | -              | Conteúdo oculto.            |
| children  | `React.ReactNode`   | `true`      | -              | -                           |
| arrow     | `boolean`           | `false`     | `true`         | -                           |
| onOpen    | `function`          | `false`     | -              | -                           |
| onClose   | `function`          | `false`     | -              | -                           |

# Icon Button

Botões permitem que os usuários tomem ações e decisões com um simples toque.

### Importação

```js
import IconButton from '@eduzz/houston-ui/IconButton';
```

### Exemplos

<Playground>
  <IconButton>
    <FaceHappyOutlineIcon />
  </IconButton>
  <IconButton disabled>
    <FaceHappyOutlineIcon />
  </IconButton>
</Playground>

### Tamanho

<Playground>
  <IconButton size='md'>
    <FaceHappyOutlineIcon />
  </IconButton>
  <IconButton size='lg'>
    <FaceHappyOutlineIcon />
  </IconButton>
</Playground>

| prop     | tipo       | obrigatório | padrão  | descrição |
| -------- | ---------- | ----------- | ------- | --------- |
| disabled | `boolean`  | `false`     | `false` | -         |
| size     | `md`, `lg` | `false`     | `lg`    | -         |

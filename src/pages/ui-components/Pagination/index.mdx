# Pagination

Componente de paginação

### Importação

```js
import Pagination from '@eduzz/houston-ui/Pagination';
```

### Exemplo

<Playground>
  {() => {
    const [page, setPage] = useState(1);
    const [perPage, setPerPage] = useState(40);
    return <Pagination page={page} perPage={perPage} total={300} onChangePage={setPage} onChangePerPage={setPerPage} />;
  }}
</Playground>

### Props

| prop            | tipo                        | obrigatório | padrão        | descrição                   |
| --------------- | --------------------------- | ----------- | ------------- | --------------------------- |
| page            | `number`                    | `true`      | -             | -                           |
| perPage         | `number`                    | `true`      | -             | -                           |
| total           | `number`                    | `true`      | -             | -                           |
| disabled        | `boolean`                   | `true`      | `false`       | -                           |
| optionsPerPage  | `string/false`              | `false`     | `10,20,40,60` | String separado por virgula |
| onChangePage    | `(page: number) => void`    | `true`      | -             | -                           |
| onChangePerPage | `(perPage: number) => void` | `true`      | -             | -                           |

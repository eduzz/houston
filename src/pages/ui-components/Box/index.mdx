# Box

O componente de caixa (Box) funciona como um encapsulador de outros componentes, auxiliando na estilização e padronização dos espaçamentos (`margin` e `padding`).

### Importação

```js
import Box from '@eduzz/houston-ui/Box';
```

### Exemplo

Padding e margin `20px`.

<Playground>
  <Box xs={{ padding: 20, margin: 20 }}>
    <div style={{ textAlign: 'center' }}>
      Lorem ipsum dolor sit amet consectetur adipisicing elit. Excepturi iusto consequuntur molestias, eligendi
      perspiciatis commodi error vero tempora voluptatum accusamus similique molestiae iste reiciendis animi, nisi
      mollitia debitis repudiandae soluta.
    </div>
  </Box>
</Playground>

### Background

Adiciona `background` no elemento.

<Playground>
  <div className='bg-grey'>
    <Box xs={{ padding: 20 }} paper>
      <div style={{ textAlign: 'center' }}>

        perspiciatis commodi error vero tempora voluptatum accusamus similique molestiae iste reiciendis animi, nisi
        mollitia debitis repudiandae soluta.
      </div>
    </Box>
  </div>
</Playground>

### Com sombra

Adiciona sombra externa no elemento.

<Playground>
  <div className='bg-grey'>
    <Box xs={{ padding: 20 }} boxShadow={2} paper>
      <div style={{ textAlign: 'center' }}>
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Excepturi iusto consequuntur molestias, eligendi
        perspiciatis commodi error vero tempora voluptatum accusamus similique molestiae iste reiciendis animi, nisi
        mollitia debitis repudiandae soluta.
      </div>
    </Box>
  </div>
</Playground>

| prop      | tipo            | obrigatório | padrão  | descrição                                                                                                       |
| --------- | --------------- | ----------- | ------- | --------------------------------------------------------------------------------------------------------------- |
| xs        | `IBoxSpacement` | `false`     | -       | Se for definido valor apenas para a propriedade `xs` todos os outros pontos de quebra vão ter os mesmos valores |
| sm        | `IBoxSpacement` | `false`     | -       | -                                                                                                               |
| md        | `IBoxSpacement` | `false`     | -       | -                                                                                                               |
| lg        | `IBoxSpacement` | `false`     | -       | -                                                                                                               |
| xl        | `IBoxSpacement` | `false`     | -       | -                                                                                                               |
| paper     | `boolean`       | `false`     | `false` | -                                                                                                               |
| boxShadow | `number`        | `false`     | -       | Aceita valores de 0 a 24                                                                                        |

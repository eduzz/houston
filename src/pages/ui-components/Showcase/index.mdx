# Showcase

O componente Showcase é utilizado para exibir informações de forma interativa ao usuário.

### Importação

```ts
import Showcase from '@eduzz/houston-ui/Showcase';
```

### Showcase

| prop        | tipo      | obrigatório | padrão | descrição                                              |
| ----------- | --------- | ----------- | ------ | ------------------------------------------------------ |
| open        | `boolean` | `true`      | -      | Controla o estado aberto/fechado do componente         |
| currentStep | `number`  | `true`      | -      | Indica em qual passo o componente será renderizado     |
| controlDots | `boolean` | `false`     | `true` | Se `true`, exibe o contador de passos totais e o atual |

### Showcase.Image

| prop | tipo     | obrigatório | padrão | descrição |
| ---- | -------- | ----------- | ------ | --------- |
| src  | `string` | `true`      | -      | -         |
| alt  | `string` | `true`      | -      | -         |

### Showcase.Title

| prop              | tipo      | obrigatório | padrão  | descrição                                                  |
| ----------------- | --------- | ----------- | ------- | ---------------------------------------------------------- |
| disableTypography | `boolean` | `false`     | `false` | Se `true`, o componente `Typography` não será renderizado. |

### Showcase.Text

| prop              | tipo      | obrigatório | padrão  | descrição                                                  |
| ----------------- | --------- | ----------- | ------- | ---------------------------------------------------------- |
| disableTypography | `boolean` | `false`     | `false` | Se `true`, o componente `Typography` não será renderizado. |

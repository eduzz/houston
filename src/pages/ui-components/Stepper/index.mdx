# Stepper

O Stepper tem como objetivo mostrar ao usuário a quantidade de etapas dentro de um fluxo, bem como indicar qualquer etapa que está sendo apresentada.

### Importação

```js
import Stepper from '@eduzz/houston-ui/Stepper';
const { Step } = Stepper;
```

### Valor controlado e prop noClick

<Playground>
  {() => {
    const { Step } = Stepper;
    const [current, setCurrent] = React.useState(0);
    const goToNext = () => {
      setCurrent(current + 1);
    };
    const goToPrev = () => {
      setCurrent(current - 1);
    };
    return (
      <Stepper current={current} noClick>
        <Step label='Básico'>
          <div>Conteúdo1</div>
          <Row justifyContent='flex-end'>
            <Column xs={3}>
              <Button onClick={goToNext}>Próximo</Button>
            </Column>
          </Row>
        </Step>
        <Step label='Personalização' description='Personalize seu produto'>
          <div>Conteúdo2</div>
          <Row justifyContent='flex-end'>
            <Column xs={3}>
              <Button onClick={goToPrev} variant='outline'>
                Anterior
              </Button>
            </Column>
            <Column xs={3}>
              <Button onClick={goToNext}>Próximo</Button>
            </Column>
          </Row>
        </Step>
        <Step label='Comunicação' description='Comunicação é muito importante'>
          <div>Conteúdo3</div>
          <Row justifyContent='flex-end'>
            <Column xs={3}>
              <Button onClick={goToPrev} variant='outline'>
                Anterior
              </Button>
            </Column>
            <Column xs={3}>
              <Button onClick={goToNext}>Próximo</Button>
            </Column>
          </Row>
        </Step>
        <Step label='Avançado'>
          <div>Conteúdo4</div>
          <Row justifyContent='flex-end'>
            <Column xs={3}>
              <Button onClick={goToPrev} variant='outline'>
                Anterior
              </Button>
            </Column>
            <Column xs={3}>
              <Button onClick={goToNext}>Finalizar</Button>
            </Column>
          </Row>
        </Step>
      </Stepper>
    );
  }}
</Playground>

### Auto gerenciável

Não há necessidade de controlar o estado do step ativo.

<Playground>

### Stepper props

| prop           | type               | required | default | description                                                                                |
| -------------- | ------------------ | -------- | ------- | ------------------------------------------------------------------------------------------ |
| onPrev         | `(number) => void` | `false`  | -       | Passa pelo callback o índice do Step                                                       |
| onNext         | `(number) => void` | `false`  | -       | Passa pelo callback o índice do Step                                                       |
| current        | `number`           | `false`  | `false` | Índice do Step atual                                                                       |
| children       | `React.ReactNode`  | `true`   | -       |                                                                                            |
| noClick        | `boolean`          | `false`  | `false` | Desabilita o click no Step                                                                 |
| destroyOnClose | `boolean`          | `false`  | `false` | Se true o componente que está dentro do Step será desmontado depois que o Step for trocado |
| mountOnEnter   | `boolean`          | `false`  | `false` | Se true o componente que está dentro do Step só será montado depois que o Step for trocado |

### Step props

| prop        | type              | required | default | description |
| ----------- | ----------------- | -------- | ------- | ----------- |
| children    | `React.ReactNode` | `false`  | -       |             |
| error       | `boolean`         | `false`  | `false` |             |
| label       | `string`          | `true`   | -       |             |
| description | `string`          | `false`  | -       |             |

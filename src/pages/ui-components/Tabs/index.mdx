# Tabs

As abas facilitam a exploração e alternam entre diferentes visualizações.

> Para as Tabs funcionarem corretamente, insira uma div encapsulando o children dentro do Tab.Content.

### Importação

```js
import Tabs from '@eduzz/houston-ui/Tabs';
```

### Auto gerenciável

Não há necessidade de controlar o estado da aba ativa.

<Playground>
  <Tabs>
    <Tabs.Content label='Título 1'>
      <div>Conteúdo 1</div>
    </Tabs.Content>
    <Tabs.Content label='Título 2'>
      <div>Conteúdo 2</div>
    </Tabs.Content>
  </Tabs>
</Playground>

### Valor controlado

<Playground>
  {() => {
    const [tab, setTab] = React.useState(0);
    return (
      <Tabs value={tab} onChange={value => setTab(value)}>
        <Tabs.Content label='Título 1'>
          <div>Conteúdo 1</div>
        </Tabs.Content>
        <Tabs.Content label='Título 2'>
          <div>Conteúdo 2</div>
        </Tabs.Content>
      </Tabs>
    );
  }}
</Playground>

#### Tabs props

| prop     | type                      | required | default | description         |
| -------- | ------------------------- | -------- | ------- | ------------------- |
| value    | `number`                  | `false`  | -       | Índice da aba.      |
| onChange | `(value: number) => void` | `false`  | -       | Recebe novo índice. |

#### Tabs.Content props

| prop            | type        | required | default |
| --------------- | ----------- | -------- | ------- |
| label           | `string`    | `true`   | -       |
| icon            | `ReactNode` | `false`  | -       |
| disabled        | `boolean`   | `false`  | `false` |
| disabledPadding | `boolean`   | `false`  | `false` |

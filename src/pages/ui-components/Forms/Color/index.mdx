# Color

O componente é responsável por exibir um campo de seleção de cor hexadecimal.

### Importação

```js
import Color from '@eduzz/houston-ui/Forms/Color';
```

### Exemplos

<Playground>
  {() => {
    const [color, setColor] = React.useState('');
    return (
      <div style={{ display: 'flex', flexDirection: 'column', flexWrap: 'wrap', gap: 16 }}>
        <Color label='Cor' value={color} onChange={setColor} />
        <Color label='Cor' loading placeholder='Selecione uma cor' value={color} onChange={setColor} />
        <Color label='Campo desabilitado' value={color} onChange={setColor} disabled />
      </div>
    );
  }}
</Playground>

### Props

Equivalente as props do componente [Input](/houston/ui-components/Forms/Input).

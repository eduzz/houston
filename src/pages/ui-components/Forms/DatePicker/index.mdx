# Date Picker

O componente é responsável por exibir um campo de data nos formulários.

### Importação

```js
import DatePicker from '@eduzz/houston-ui/Forms/DatePicker';
```

### Exemplo

<Playground>
  {() => {
    const [value, setValue] = React.useState(null);
    return (
      <Container>
        <Row>
          <Column sm={12} md={4}>
            <DatePicker label='Data' value={value} onChange={value => setValue(value)} />
          </Column>
          <Column sm={12} md={4}>
            <DatePicker label='Data e Hora' mode='datetime' value={value} onChange={value => setValue(value)} />
          </Column>
          <Column sm={12} md={4}>
            <DatePicker label='Hora' mode='time' value={value} onChange={value => setValue(value)} />
          </Column>
        </Row>
      </Container>
    );
  }}
</Playground>

| prop          | tipo                 | obrigatório | padrão  | descrição                                                                           |
| ------------- | -------------------- | ----------- | ------- | ----------------------------------------------------------------------------------- |
| id            | `string`             | `false`     | -       | -                                                                                   |
| name          | `string`             | `true`      | -       | -                                                                                   |
| label         | `string`             | `false`     | -       | -                                                                                   |
| className     | `string`             | `false`     | -       | -                                                                                   |
| placeholder   | `string`             | `false`     | -       | -                                                                                   |
| disabled      | `boolean`            | `false`     | `false` | -                                                                                   |
| fullWidth     | `boolean`            | `false`     | `true`  | -                                                                                   |
| loading       | `boolean`            | `false`     | -       | Exibe ícone de carregamento no componente de texto.                                 |
| helperText    | `string`             | `false`     | -       | Exibe texto de ajuda abaixo do componente de texto.                                 |
| errorMessage  | `string`             | `false`     | -       | Exibe uma mensagem de erro no componente de texto, assim como indica erro no campo. |
| value         | `any`                | `false`     | -       | -                                                                                   |
| onChange      | `function`           | `false`     | -       | Função disparada ao alterar o valor do campo de texto.                              |
| mode          | `date/datetime/time` | `false`     | `date`  | -                                                                                   |
| displayFormat | `string`             | `false`     | -       | Formatos suportados pela biblioteca `date-fns`.                                     |
| maxDate       | `Date`               | `false`     | -       | -                                                                                   |
| minDate       | `Date`               | `false`     | -       | -                                                                                   |

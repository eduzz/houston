# Radio

O componente de Radio é utilizado quando é preciso deixar todas as opções visíveis e permitir a seleção delas.

### Importação

```js
import Radio from '@eduzz/houston-ui/Forms/Radio';
```

### Exemplo com Form

<Playground>
  {() => {
    const form = useForm({ onSubmit: () => {} });
    return (
      <Form context={form}>
        <p>{JSON.stringify(form.values)}</p>
        <Radio name='value' value={1}>
          Opção 1
        </Radio>
        <Radio name='value' value={2}>
          Opção 2
        </Radio>
      </Form>
    );
  }}
</Playground>

### Exemplo sem Form

<Playground>
  {() => {
    const [value, setValue] = React.useState();
    return (
      <div>
        <p>{JSON.stringify({ value })}</p>
        <Radio helperText='Descrição do item' onChange={setValue} value={1} checked={value}>
          Opção 1
        </Radio>
        <Radio errorMessage='Mensagem de erro' onChange={setValue} value={2} checked={value}>
          Opção 2
        </Radio>
      </div>
    );
  }}
</Playground>

### Props

| prop         | tipo       | obrigatório | padrão  | descrição                                            |
| ------------ | ---------- | ----------- | ------- | ---------------------------------------------------- |
| name         | `string`   | `false`     | -       |                                                      |
| helperText   | `string`   | `false`     | -       |                                                      |
| errorMessage | `string`   | `false`     | -       |                                                      |
| value        | `boolean`  | `false`     | -       | Valor setado caso seja checkado                      |
| checked      | `boolean`  | `false`     | `false` | Pode receber um boolean ou um state com o valor      |
| onChange     | `function` | `false`     | -       | Função chamada quando o valor do checkbox é alterado |
| disabled     | `boolean`  | `false`     | -       |                                                      |

# Search

O componente de pesquisa é um campo de busca e permite que o usuário encontre um conteúdo de forma rápida.

### Importação

```js
import Search from '@eduzz/houston-ui/Forms/Search';
```

### Exemplo

<Playground>
  {() => {
    const results = React.useMemo(() => Array.from({ length: 30 }, (_, i) => `Resultado ${i + 1}`), []);
    const [text, setText] = React.useState('');
    const filterStrategy = React.useCallback(result => result.toLowerCase().includes(text.toLowerCase()), [text]);
    return (
      <div style={{ display: 'flex', flexWrap: 'wrap', gap: 16 }}>
        <Search label='Pesquise pelos resultados' value={text} onChange={setText}>
          {results.map(result => (
            <Search.Result key={result}>{result}</Search.Result>
          ))}
        </Search>
        <Search
          label='Resultados com estratégia de filtro'
          value={text}
          onChange={setText}
          filterStrategy={filterStrategy}
        >
          {results.map(result => (
            <Search.Result key={result}>{result}</Search.Result>
          ))}
        </Search>
        <Search label='Desativado' disabled value={text} onChange={setText}>
          {results.map(result => (
            <Search.Result key={result}>{result}</Search.Result>
          ))}
        </Search>
        <Search label='Carregando' loading value={text} onChange={setText}>
          {results.map(result => (
            <Search.Result key={result}>{result}</Search.Result>
          ))}
        </Search>
        <Search label='Com texto de ajuda' helperText='Exemplo de texto de ajuda' value={text} onChange={setText}>
          {results.map(result => (
            <Search.Result key={result}>{result}</Search.Result>
          ))}
        </Search>
        <Search label='Com mensagem de erro' errorMessage='Mensagem de erro' value={text} onChange={setText}>
          {results.map(result => (
            <Search.Result key={result}>{result}</Search.Result>
          ))}
        </Search>
      </div>
    );
  }}
</Playground>

### Search Props

| prop           | tipo                                                                | obrigatório | padrão | descrição                                                                                                     |
| -------------- | ------------------------------------------------------------------- | ----------- | ------ | ------------------------------------------------------------------------------------------------------------- |
| onChange       | `(value: string, e?: React.ChangeEvent<HTMLInputElement>) => any`   | `false`     | -      |                                                                                                               |
| onSelect       | `(value: string, e?: React.MouseEvent<HTMLDivElement>) => any`      | `false`     | -      |                                                                                                               |
| onEnter        | `(value: string, e?: React.KeyboardEvent<HTMLInputElement>) => any` | `false`     | -      |                                                                                                               |
| filterStrategy | `(result: string, index?: number, results?: string[]) => boolean`   | `false`     | -      | Função para filtrar os resultados informados, para que um resultado seja exibido, a função deve retornar true |

E as props do componente [Input](/houston/ui-components/Forms/Input).

### Search.Result Props

| prop      | tipo     | obrigatório | padrão | descrição |
| --------- | -------- | ----------- | ------ | --------- |
| children  | `string` | `true`      | -      | -         |
| className | `string` | `false`     | -      |           |

# Date Range Picker

O componente é responsável por exibir campos parar selecionar um período de data nos formulários.

### Importação

```js
import DateRangePicker from '@eduzz/houston-ui/Forms/DateRangePicker';
```

### Exemplo

<Playground>
  {() => {
    const [value, setValue] = React.useState(null);
    return (
      <Container>
        <Row>
          <Column sm={12} md={4}>
            <DateRangePicker label='Período' value={value} onChange={value => setValue(value)} />
          </Column>
          <Column sm={12} md={4}>
            <DateRangePicker
              label='Período com data e hora'
              mode='datetime'
              value={value}
              onChange={value => setValue(value)}
            />
          </Column>
          <Column sm={12} md={4}>
            <DateRangePicker label='Período de hora' mode='time' value={value} onChange={value => setValue(value)} />
          </Column>
        </Row>
      </Container>
    );
  }}
</Playground>

# Switch

Switches alternam o estado de uma única configuração, sendo ligado ou desligado.

### Importação

```js
import Switch from '@eduzz/houston-ui/Forms/Switch';
```

### Exemplos

<Playground>
  {() => {
    const [checked, setChecked] = React.useState(false);
    const onChange = React.useCallback(() => setChecked(prev => !prev), []);
    return (
      <Row>
        <Column xs='auto'>
          <Switch value={checked} onChange={onChange}>
            Switch
          </Switch>
        </Column>
        <Column xs='auto'>
          <Switch value disabled>
            Checked disabled
          </Switch>
        </Column>
        <Column xs='auto'>
          <Switch disabled>Disabled</Switch>
        </Column>
        <Column xs='auto'>
          <Switch errorMessage='Error Message' value={checked} onChange={onChange}>
            Error
          </Switch>
        </Column>
      </Row>
    );
  }}
</Playground>

### Utilizando com Form

```js
import useForm from '@eduzz/houston-forms/useForm';
```

<Playground>
  {() => {
    const form = useForm({ defaultValues: { switch: true } });
    const value = form.watch('switch');
    return (
      <Form context={form} onSubmit={() => {}}>
        <Typography>Valor atual do switch: {String(value)}</Typography>
        <br />
        <Switch name='switch' />
      </Form>
    );
  }}
</Playground>

| prop         | tipo       | obrigatório | padrão  | descrição                                 |
| ------------ | ---------- | ----------- | ------- | ----------------------------------------- |
| name         | `string`   | `true`      | -       | Necessário quando usado dentro de um Form |
| value        | `boolean`  | `false`     | `false` | -                                         |
| onChange     | `function` | `false`     | -       | -                                         |
| disabled     | `boolean`  | `false`     | `false` | -                                         |
| errorMessage | `string`   | `false`     | -       | -                                         |

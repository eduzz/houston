# Checkbox

O componente de caixa de seleção (Checkbox) permite que o usuário selecione um ou mais itens de um determinado grupo.

### Importação

```js
import Checkbox from '@eduzz/houston-ui/Forms/Checkbox';
```

### Variações

<Playground>
  {() => {
    const [checked, setChecked] = React.useState(false);
    return (
      <>
        <Checkbox checked={checked} onChange={setChecked}>
          Simples
        </Checkbox>
        <Checkbox helperText='Exemplo de utilização do Checkbox com descrição' checked={checked} onChange={setChecked}>
          Com descrição
        </Checkbox>
        <Checkbox errorMessage='Mensagem de erro' checked={checked} onChange={setChecked}>
          Com erro
        </Checkbox>
        <Checkbox disabled helperText='Checkbox sem margem' checked={checked} onChange={setChecked}>
          Desativado
        </Checkbox>
      </>
    );
  }}
</Playground>

### Utilizando método onChange (sem hook useForm)

<Playground>
  {() => {
    const [value, setValue] = React.useState(false);
    const handleChange = checked => {
      setValue(checked);
      console.log('current checkbox value', checked);
    };
    return (
      <Checkbox helperText='Checkbox utilizando método de onChange' onChange={handleChange} checked={value}>
        Exemplo com método onChange
      </Checkbox>
    );
  }}
</Playground>

### Props

| prop         | tipo       | obrigatório | padrão  | descrição                                                  |
| ------------ | ---------- | ----------- | ------- | ---------------------------------------------------------- |
| name         | `string`   | `false`     | -       |                                                            |
| helperText   | `string`   | `false`     | -       |                                                            |
| errorMessage | `string`   | `false`     | -       |                                                            |
| value        | `boolean`  | `false`     | -       | Valor setado caso seja checkado                            |
| multiple     | `boolean`  | `false`     | `false` | Trata o valor passado como um array, útil em caso de lista |
| checked      | `boolean`  | `false`     | `false` | Pode receber um boolean ou um array no caso de multiple    |
| onChange     | `function` | `false`     | -       | Função chamada quando o valor do checkbox é alterado       |
| disabled     | `boolean`  | `false`     | -       |                                                            |

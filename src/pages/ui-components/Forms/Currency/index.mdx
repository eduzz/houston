# Currency

O componente é responsável por formatar campos monetários em formulários.

### Importação

```js
import CurrencyField from '@eduzz/houston-ui/Forms/Currency';
```

### Exemplos

<Playground>
  {() => {
    const [money, setMoney] = React.useState('');
    return <CurrencyField label='Valor' value={money} onChange={setMoney} />;
  }}
</Playground>

### Props

Equivalente as props do componente [Input](/houston/ui-components/Forms/Input).

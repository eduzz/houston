# Link

O componente Link é usado usado para navegação.

### Importação

```js
import Link from '@eduzz/houston-ui/Link';
```

### Exemplo com componentes externos (Link, NavLink, ...).

```jsx
import { NavLink } from 'react-router-dom';

<Link as={NavLink} to='/home'>
  Home
</Link>;
```

### Link props

| prop     | tipo                | obrigatório | padrão    |
| -------- | ------------------- | ----------- | --------- |
| children | `React.ReactNode`   | `true`      | -         |
| as       | `React.ElementType` | `false`     | `a`       |
| href     | `string`            | `false`     | -         |
| showIcon | `boolean`           | `false`     | -         |
| size     | `LinkSizes`         | `false`     | `inherit` |

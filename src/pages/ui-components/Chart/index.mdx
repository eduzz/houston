# Showcase

Gráficos utilizados para apresentação de dados de forma comparativa, ilustrados de diferentes formas (barras, pizza,
linhas e donut).

### Importação

```ts
import Chart from '@eduzz/houston-ui/Chart';
```

### Chart

| prop   | tipo                                                         | obrigatório | padrão | descrição                      |
| ------ | ------------------------------------------------------------ | ----------- | ------ | ------------------------------ |
| data   | `{ name: string; bars: { label: string, value: number }[] }` | `true`      | -      | Popula o gráfico com dados     |
| height | `number`                                                     | `true`      | -      | Determina a altura do gráfico  |
| width  | `number`                                                     | `true`      | -      | Determina a largura do gráfico |
| color  | `{ [key: string]: string }`                                  | `true`      | -      | Determina as cores das colunas |

### Chart.Bar

| prop   | tipo                                                         | obrigatório | padrão | descrição                      |
| ------ | ------------------------------------------------------------ | ----------- | ------ | ------------------------------ |

| width  | `number`                                                     | `true`      | -      | Determina a largura do gráfico |

# Button (deprecated)

Utilizar o Button do AntD (https://ant.design/components/button)

## Como migrar

Diferenças de implementação:

- Se não tiver variant colocar **type="primary"**.
- Utilizar os icons do próprio AntD (https://ant.design/components/icon)

```diff
- import Button from '@eduzz/houston-ui/Button'
+ import { Button } from antd

 <Button
-  startIcon={...}
+  icon={...}
-  type='submit'
+  htmlType='submit'
-  variant='contained'
+  type='primary' // contained=primary outlined=default text=text

-  fullWidth
-  endIcon
-  active
 />

```

| Variant   | AntD Type |
| --------- | --------- |
| contained | primary   |
| outlined  | default   |
| text      | text      |
